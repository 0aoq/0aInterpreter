# repeat

| **Deprecation Warning:** `repeat` is deprecated. Please use `for` instead. |
|:-:|

Loop code every `x` seconds `y` times

- actions: **(0a)**
- every: **(number)**
- i: **(number)**

```
repeat {/s} &;actions {/set} &;every i &;i
```

See: [function](https://0aoq.github.io/0aInterpreter/?md/api/keywords/func.md)
